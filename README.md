# 🌿 EcoGuardian Backend

AI-Powered Industrial Sustainability Operating System

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- MongoDB
- Redis
- MQTT Broker

### Installation
```bash
# Clone repository
git clone https://github.com/your-org/ecoguardian-backend.git
cd ecoguardian-backend

# Install dependencies
npm install

# Setup environment
cp .env.example .env
# Edit .env with your configuration

# Start development server
npm run dev
